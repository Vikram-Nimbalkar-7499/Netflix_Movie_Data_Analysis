# 🎬 Netflix Movie Data Analysis (EDA)

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on a Netflix movies dataset to uncover insights into genre distribution, movie popularity, and yearly trends.

The analysis focuses on transforming raw data into meaningful insights that help understand audience preferences and content trends.

---

## 🎯 Objectives

* Identify the most frequent genres
* Analyze popularity and vote trends
* Discover highest and lowest rated movies
* Understand yearly movie release patterns

---

## 🧹 Data Cleaning & Preprocessing

* Removed missing and null values
* Handled inconsistent genre formats
* Converted data types (dates, numerical fields)
* Removed duplicates for accurate analysis

---

## 📊 Data Visualization

Used visual techniques to uncover patterns:

* Genre distribution charts
* Popularity vs vote count graphs
* Year-wise movie release trends
* Bar plots, count plots, and heatmaps

Libraries used:

* Matplotlib
* Seaborn

---

## 📈 Key Insights

### 🔹 Most Frequent Genre

* **Drama** is the most frequent genre
* Appears in more than **14%** of the dataset

### 🔹 Genres with Highest Votes

* Around **25.5%** of movies have high vote counts
* **Drama** leads again with over **18.5% popularity share**

### 🔹 Highest Popular Movie

* 🎥 *Spider-Man: No Way Home*
* Genres: Action, Adventure, Science Fiction

### 🔹 Lowest Popular Movie

* 🎥 *The United States, Thread*
* Genres: Music, Drama, War, Sci-Fi, History

### 🔹 Year with Most Movies

* 📅 **2020** has the highest number of movies released
